# Water Sort Puzzle

A mobile-first Water Sort Puzzle game built using PixiJS 8, TypeScript, GSAP, Howler, and Vite.

## Features
- **Mobile First & Responsive**: Optimized for portrait mobile screens (1080x1920 design resolution) with dynamic scaling.
- **60 FPS Performance**: Hardware-accelerated rendering, object pooling, and 60 FPS GSAP animations.
- **20 Predefined Levels + Infinite Procedural Levels**: Guaranteed 100% solvable level generation algorithm.
- **Daily Challenge & Random Play**: Date-seeded daily puzzle with +100 coin reward.
- **Full Game Logic Isolation**: Game model is 100% independent from rendering engines.
- **Zero-Asset Web Audio Synthesizer**: Web Audio API oscillator synthesis for instant tap, pour, click, and win sounds.
- **Progression & Economy**: Hint solver, multi-step undo system, star ratings, and coin earnings saved via `LocalStorage`.
- **Abstracted Ads & Analytics**: Ready for native AdMob and Firebase Analytics SDK integration.

## Getting Started

### Development
```bash
npm install
npm run dev
```

### Production Build
```bash

```
